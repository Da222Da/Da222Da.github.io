## 什么是面向对象编程？

在搞清楚什么是面向对象之前，先得搞清楚以下几个概念：

### 1.类和实例

- 类，即是模板。`例如，张三是人、旺财是狗。这里的人、狗就是我们所说的类`。
- 实例，类的具体化。`例如，张三也算是个人。这里的张三就是一个实例`。

```js
// 声明人类
class People {
  constructor(name) {
    this.name = name;
  }
}

// 声明狗类
class Dog {
  constructor(name) {
    this.name = name;
  }
}

let p1 = new People("张三"); // 声明人类的实例张三
let a1 = new Dog("旺财"); // 声明狗类的实例旺财
```

### 2.类的三大特性：继承、封装、和多态

- 什么是继承？继承，子类继承父类。
- 什么是封装？封装，数据的权限和保密。
- 什么是多态？多态，同一接口的不同实现。
